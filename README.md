Real-Time Currency Converter with GUI


Overview

This project is a real-time currency converter application with a graphical user interface (GUI). 
It fetches live exchange rates from an external API and allows users to convert currencies easily through an interactive interface.


Features

Real-time currency conversion
Interactive GUI built using Tkinter
Fetches live exchange rates using API
User-friendly input and output display
Supports multiple currencies


Tech Stack

Python
Tkinter (GUI)
API Integration
Requests library


How It Works

User enters amount and selects currencies via GUI
Application sends request to currency exchange API
Receives live exchange rate data
Displays converted result instantly in the interface


Future Improvements

Improve GUI design and styling
Add historical exchange rate visualization
Deploy as a desktop or web application
